# AICourseNote

一个面向网课与课程学习场景的 AI 辅助桌面笔记应用，支持富文本记录、图片整理、PDF 导出、系统音频转写和 AI 分析，帮助把零散课堂内容沉淀成更易复习的学习资料。

## 项目简介

在课程学习过程中，信息往往分散在课堂音频、讲义截图、临时草稿和课后总结之间。AICourseNote 希望把这些动作集中到一个本地桌面应用中，让“记录、整理、导出、复习”形成一条更顺手的工作流。

项目当前围绕以下场景展开：

- 记录课程笔记并持续编辑整理
- 插入课堂截图、图片和结构化内容
- 导出当前草稿为 PDF，便于归档与分享
- 结合系统音频转写和 AI 分析辅助课后整理
- 在本地管理数据，并通过系统安全存储保存 API key

## 主要功能

### 课程与笔记管理

- 支持课程笔记本的新建、重命名、删除与列表管理
- 支持笔记创建、编辑、删除，并按更新时间排序
- 三栏桌面布局清晰区分课程、笔记与编辑区

### 富文本编辑

- 基于 TipTap 构建富文本编辑体验
- 支持高亮、文字颜色、字号、代码块、表格等常用能力
- 支持插入本地图片，适合整理课件截图与图示内容
- 保存前会进行 HTML 白名单净化，降低脏内容落库风险

### 导出与整理

- 支持将当前草稿直接导出为 PDF
- 导出时会处理本地图片，减少 PDF 丢图问题
- 适合用于课程整理、复习归档和阶段性输出

### 转写与 AI 辅助

- 支持系统音频转写，适合网课或课程回放记录
- 支持 AI 分析结果查看、删除和另存为新笔记
- AI 结果与原始笔记内容分离，不会自动覆盖正文

### 本地优先与数据安全

- 本地保存课程数据、笔记内容与图片资源
- API key 通过系统安全存储管理，不直接写入数据库
- 设置页可查看当前数据库路径与图片目录路径

## 本地开发

安装依赖：

```bash
npm install
```

启动开发模式：

```bash
npm run dev
```

构建并本地启动：

```bash
npm run build
npm run start
```

类型检查：

```bash
npm run typecheck
```

## 打包命令

生成 Windows 便携版：

```bash
npm run dist:win
```

生成目录版程序：

```bash
npm run dist:dir
```

生成本地安装包：

```bash
npm run dist:local
```

默认打包输出目录为 `release/`。
