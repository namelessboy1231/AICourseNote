使用方式：

1. 将整个目录解压到你想安装 AICourseNote 的目标位置。
2. 双击 Install-AICourseNote.cmd。
3. 安装完成后，运行 AICourseNote\AICourseNote.exe。
4. 如需彻底删除程序与数据，双击 Uninstall-AICourseNote.cmd。

说明：

- 安装完成后，payload\ 安装载荷目录会自动删除，避免同目录保留两份程序文件。
- 卸载会删除程序目录、data\ 数据目录，以及系统安全存储中的 AICourseNote API key。

目录说明：

- AICourseNote\：程序本体与运行时依赖
- data\：数据库、图片、日志等本地数据
- payload\：安装前临时载荷目录，安装成功后会自动清理
